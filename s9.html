S004-Virtual

Para comenzar la clase hicimos un repaso de < div > y < span >, estas etiquetas son fundamentales para entender y comenzar con CSS.
Aprendimos que CSS mira lo que esta en HTML mediante cajas

Existen dos formas de agregar CSS a tu HTML y son

  1. Link: Que funciona una pagina aparte solo para el CSS
  2. Style: Que agrega el CSS en la misma pagina. Esto funciona para agregar estilo breve ya que si se extiende puede llegar a desordenar tu código
